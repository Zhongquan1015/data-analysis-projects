# Data Analysis Projects

This repository contains selected data analysis and data science projects completed during coursework and internships.

The projects focus on statistical modeling, causal thinking, experiments, and practical data analysis using Python and R.  
Original datasets are not included due to privacy or course policy constraints.

---

## Projects Overview

### 1. Boston Housing Price Analysis
- Objective: Analyze key socioeconomic and environmental factors affecting housing prices.
- Methods: Exploratory data analysis, linear regression, Box-Cox transformation, feature selection, WLS, Random Forest, XGBoost.
- Outcome: Built interpretable regression models and compared them with machine learning approaches.

### 2. Telecom User Behavior Prediction
- Objective: Predict user income and behavioral attributes using telecom user data.
- Methods: Data preprocessing, feature engineering, regression and classification models (Logistic Regression, Random Forest, XGBoost, SVM).
- Evaluation: RMSE, MAE, Precision, Recall, F1-score, AUC.

### 3. Metro Travel Pattern Clustering
- Objective: Identify typical metro travel patterns at both trip-level and passenger-level.
- Methods: Feature construction, similarity analysis, K-means, Hierarchical Clustering, DBSCAN.
- Outcome: Interpreted clusters and summarized representative travel behaviors.
